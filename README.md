# 量化实验

## 项目概述
SUDA金融+计算机，大四上，大数据与量化投资因子实验

## 使用方法
### 0. 把这个项目clone到你本地

### 1. 安装依赖
在终端中运行以下命令安装所需依赖：
```bash
pip install baostock pandas numpy matplotlib
```

### 2. 配置回测时间（重要）
在 `data_gen.py` 文件中配置回测时间

### 3. 生成数据
运行数据生成脚本：`exp/data_gen.py`

该脚本将生成沪深300、中证500和中证800成分股的历史数据，并保存到 `data` 文件夹中。

### 4. 运行回测
运行主实验文件：`exp/exp.py`

### 5. 查看结果
回测结果保存在 `res/` 文件夹中，每个因子都有一个独立的文件夹，包含实验报告上的所有东西

## 注意事项
1. **数据获取限制**：`baostock` 的数据获取，每隔一段时间会停止一会，耐心等待就行
2. **因子计算与回测**：部分因子可能因数据不足或计算问题导致回测结果为空，需根据具体因子调整或优化。
